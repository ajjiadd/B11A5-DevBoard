Question-1: What are the different ways to select an element in the DOM?

The Document Object Model (DOM) is the lifeblood of a website. If we compare it to the human body, the skeleton is the structure such as HTML, the skin is CSS, and the lifeblood is JavaScript or DOM.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML: This returns everything inside an element (including HTML tags).
  • innerText: This only takes visible text, ignoring hidden elements. It does not process HTML.
  • textContent: This is similar to innerText, but takes all text, even hidden text. It works faster because it does not check the style. If you only need the text and style or visibility are not important, this is the best option.

Question-3: What is event delegation in the DOM?

Event Delegation means placing events on the parent element and handling events on child elements. This requires less code, the event still works even when a new element is added, and the performance is good.

Question-4: What is event bubbling in the DOM?

Event bubbling directs an event to its target. It works like this: When an element (like a button) is clicked, an event is directed to the element. If an event handler is set for the element, the event handler is triggered. Then the event "bubbles up" to the elements parent.

                    let parentDiv = document.getElementById('parent');
